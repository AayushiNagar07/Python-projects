## 🚀 Problem Statement

Teachers spend too much time manually creating question papers.

## 💡 Solution

GenQAI automates question paper generation using structured question banks and templates.

## 📊 Status

MVP in development (Streamlit-based SaaS prototype)

## 🧠 Architecture

User → Streamlit UI → Question Bank (JSON) → Generator Engine → PDF Output

## 🔮 Future Features

- AI-based question generation (LLM integration)
- Multi-teacher SaaS system
- Cloud database (PostgreSQL)
- Mobile app version
